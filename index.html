Basic example

It transforms this
Into this

Code

$('#basic').editableSelect();

Effects

Default
Slide
Fade

Code

$('#default').editableSelect({ effects: 'default' });
$('#slide').editableSelect({ effects: 'slide' });
$('#fade').editableSelect({ effects: 'fade' });

Dialog

Open modal dialog

Modal title

Code

$('#appendTo').editableSelect({ appendTo: '.modal-body' });

No filter

Code

$('#filter').editableSelect({ filter: false });

HTML support

Code

$('#html').editableSelect();

onSelect

Last selected:

Code

$('#onselect').editableSelect({
	onSelect: function (element) {
		$('#afterSelect').html($(this).val());
	}
});
